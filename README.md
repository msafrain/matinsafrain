## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `matinsafrain`.
2. Upload all files from this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Save and wait for the GitHub Pages URL to become available.
6. In the same Pages settings, enter `matinsafrain.com` under **Custom domain**.
7. Configure the domain's DNS records as described in GitHub's custom-domain guide.
8. When the DNS check succeeds, enable **Enforce HTTPS**.
